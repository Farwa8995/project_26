Fusion Theme Options

CONTROL YOUR SITE GLOBALLY

The Fusion Theme Options control options and settings globally throughout the site. Set it in one place, and it is applied everywhere. Theme Options
interact with every Design Element and layout on the site so you can quickly modify something and then see it populate across every page and post.

Fusion Page Options

CONTROL INDIVIDUAL PAGES & POSTS

These options extend the global Theme Options, allowing you to customize individual
pages/posts that will be unique and stand out from the rest of your website. This control is all
achieved by overriding the global options set in the Theme Options.
